== Description ==

This plugin will help you to create and manage tabs. Competible with any themes. Just need to use simple short-code [quick-tabs tab_group=id] to insert tabs in page/post content

= Features: =
*	Managable tabs
*	Shortcode support inside tab content
*	Managable tab groups
*	Same tab usable with multiple tab groups
*	Easy option for addiing/editing tab in tab group
*	Tiny-mce button for inserting tabs inside post/page content
*	Nice looking jquery tab style.
*	Competible with any wp theme.
*	Coming more on next update


== Installation ==


1. Upload `quick-tabs` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
